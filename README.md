# EasyToVibe 🌊

EasyToVibe（イージートゥバイブ）は、プロジェクトのディレクトリ構造とソースコードを指定のルールに基づいて1つのMarkdownファイルに瞬時に統合・集約するVS Code拡張機能です。

大規模なコンテキストウィンドウ（最大200万トークン）を持つ無料版GeminiなどのAIエージェントと、ノーストレスで「バイブコーディング（ペアプログラミング）」を楽しむために開発されました。

## 特徴

- 📂 **スマートなツリー生成**: プロジェクト全体のディレクトリ構造をテキストツリー形式で出力。
- 🚫 **`.gitignore` を完全リスペクト**: `node_modules` などの不要なファイルは自動で除外されます。
- 📦 **特定フォルダのコンテンツスキップ**: ツリー構造には名前を残しつつ、中身の解析をスキップする高度なフィルタリング。
- 📑 **自動エディタオープン**: 生成されたMarkdownファイル（`markdowns/project_context_*.md`）を自動でVS Code内に開き、すぐにコピー＆ペーストが可能です。

## 使い方

1. 拡張機能をインストールし、対象のワークスペース（プロジェクト）を開きます。
2. コマンドパレット（`Ctrl+Shift+P` または `Cmd+Shift+P`）を開きます。
3. `AI用: プロジェクトの全コードをコピー` を実行します。
4. プロジェクトの解析が始まり、自動的に `markdowns/` ディレクトリ内にMarkdownファイルが生成され、エディタに表示されます。
